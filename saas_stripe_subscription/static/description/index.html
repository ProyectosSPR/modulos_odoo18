SaaS Stripe Subscription Integration

Synchronize Odoo Subscriptions with Stripe

Automatically sync your Odoo subscriptions with Stripe's subscription API. Create, update, and cancel subscriptions seamlessly between both platforms.

Features

Automatic Subscription Sync

  • Create subscriptions in Stripe when started in Odoo
  • Cancel subscriptions in Stripe when closed in Odoo
  • Update subscription items when products change
  • Sync subscription status from Stripe

Product Management

  • Automatic product creation in Stripe
  • Price synchronization with Stripe
  • Recurring interval mapping
  • Support for test and production modes

How It Works

  1. Configure your Stripe payment provider with API keys
  2. Enable Stripe sync in subscription settings
  3. Create subscriptions in Odoo as usual
  4. When you start a subscription, it's automatically created in Stripe
  5. Any changes to products or quantities sync to Stripe
  6. Closing the subscription cancels it in Stripe

Test & Production Ready

The module automatically detects test vs production mode based on your Stripe payment provider configuration. Use test API keys for development and live keys for production - no code changes needed!
